# 🚀 Team Task Manager (Full-Stack)

A full-stack web application to manage tasks with authentication and role-based access.

---

## 📌 Features

- 🔐 User Authentication (Signup/Login)
- 📋 Create, update, and delete tasks
- 📊 Task status tracking (Pending / In Progress / Completed)
- 🌐 REST API integration
- 📦 MongoDB database
- ⚡ Full-stack (React + Node.js)

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Bcrypt
